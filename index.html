🔎What is OpenClaw?

OpenClaw is a community-driven, open-source project that recreates the beloved 1997 platformer Captain Claw — originally developed by Monolith Productions and published by GT Interactive. The project reverse-engineers and reimplements the game engine, allowing players to experience the original levels with full compatibility on modern operating systems.

Captain Claw features a swashbuckling anthropomorphic cat pirate who fights his way through 14 imaginative levels filled with enemies, treasures, and boss battles. OpenClaw preserves every detail of the original experience while making it accessible to new audiences.

Did you know?

The original Claw was released as freeware in 2002 by Monolith Productions, making it legally available to play. OpenClaw builds on this legacy by providing an open engine that runs the original game assets natively on Linux, macOS, and Windows.

Core Technology Stack

SDL2 — Window management, rendering, input, and audio mixing

Box2D — Physics simulation for accurate collision and movement

TinyXML2 — Parsing of game configuration and level data

SDL2_mixer — MIDI/WAV audio playback matching original game feel

Building from Source

# Clone the repository
git clone https://github.com/Piotr-Pht-Jasiek/OpenClaw.git
cd OpenClaw

# Install dependencies (Ubuntu/Debian)
sudo apt install cmake libsdl2-dev libsdl2-mixer-dev \
     libsdl2-image-dev libbox2d-dev

# Build
mkdir build && cd build
cmake ..
make -j$(nproc)

🕐History

1997 — Original Claw Released

Monolith Productions releases Captain Claw for Windows 95. It becomes a cult classic praised for its detailed sprite art and tight platformer gameplay.

2002 — Released as Freeware

Monolith makes Claw freely available for download, breathing new life into the community and enabling legal redistribution of the game assets.

2015 — OpenClaw Project Begins

Community developers begin reverse-engineering the original engine and building an open-source reimplementation to bring Claw to modern platforms.

Ongoing — Community Development

OpenClaw continues to receive contributions from fans worldwide, improving accuracy, fixing bugs, and expanding platform support.
